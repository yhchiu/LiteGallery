一個 Android 的相簿 App，具備以下功能：

- 名稱是 LiteGallery
- 以速度為最高考量
- compileSdk 與 targetSdk 設定為 36
- namespace 與 applicationId 設定為 "com.litegallery"
- UI 美觀大方
- App 打開後會看到所有包含照片或影片的資料夾, 點進資料夾後會看到縮圖、簡潔列表或詳細列表(可選)
- 照片或影片預設以檔案日期排序 (可改為檔名排序，可選降冪升冪)
- 觀看相片與影片，預設以全螢幕顯示，點一下會顯示操作 UI
- 操作 UI：
  - 螢幕上方顯示檔名與 … 按鈕
    - 檔名如果太長則用滾動的方式來顯示，可點一下檔名，以 toast 顯示全名
  - 螢幕下方是動作列，動作列按鈕可以自訂順序與要顯示哪些按鈕
- 可用的動作按鈕如下：
  - 旋轉螢幕
  - 刪除
  - 更改主檔名
    - 記得之前新增過的字串，可用下拉式選單選擇後附加到輸入框內目前主檔名後面
    - 請注意影片改名後，不要影響影片觀看以及進度條功能
  - 屬性(顯示檔名、大小、日期、EXIF…等資訊)
  - 分享：ACTION_SEND 分享當前媒體。
  - 編輯檔案：呼叫外部編輯器（ACTION_EDIT）
  - 修改相片旋轉方向
  - 複製
  - 移動
- 複製、移動、更改檔名前先檢查，若檔名衝突時詢問使用者如何處理：取消、覆蓋、使用新名字(預設「檔名 (1).ext」、「檔名 (2).ext」)
- 觀看影片時，離開全螢幕顯示操作 UI 時，畫面下方(動作列上方)會有 暫停/播放+進度條+左箭頭+右箭頭+下箭頭。左/右箭頭提供逐幀前進/後退功能。點 ”下箭頭” 會展開出現另外兩個水平 scrollbar，分別提供亮度與對比度的調整(API 30 以下不提供影片亮度與對比度調整,不顯示進階滑桿)
- 觀看相片與影片時支援以下手勢
  - 手勢左右滑動可以切換同資料夾上一個、下一個檔案
  - 手勢在螢幕左側邊緣滑動可以調整螢幕亮度 (退出 App 時應恢復原本的值)
  - 手勢在螢幕右側邊緣滑動可以調整音量  (退出 App 時應恢復原本的值)
  - 用兩指手勢進行縮放
  - 照片點兩下快速放大縮小(1x/2x/3x), 影片點兩下的動作暫停/播放
- App 註冊為可以處理特定(MIME type (image/* 代表所有圖片，video/* 代表所有影片) 的「意圖處理器」，讓其他 App 在點擊影片或照片時，可以呼叫此 App來開啟
  - 被其他 App 呼叫觀看相片與影片時，不管該檔案是不是在媒體資料夾，不管是否存在 .nomedia，都必須支援手勢左右滑動切換同資料夾上一個、下一個檔案
- 設定頁：
  - 自訂動作列順序與顯示哪些按鈕
  - 預設排序方式
  - 其他可供設定的選項
- 支援垃圾桶功能
  - 刪除時預設移到垃圾桶，但提供 checkbox 可直接刪除
  - 可觀看垃圾桶，恢復檔案
  - 可清空垃圾桶
- 多國語文功能，提供英文、繁體中文、簡體中文翻譯
